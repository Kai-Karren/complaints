# complaints

## Usage

### Install 

```bash
pip install git+ssh://git@github.com/Kai-Karren/complaints.git
```

## Install from Source

### Requirements

```bash
pip install build
```

### Build

```bash
python -m build
```

### Install

```bash
pip install --force-reinstall  dist/complaints-0.0.1-py3-none-any.whl
```
